# nextjs-app-passport-demo

This is a demonstration repository for [nextjs-app-passport](https://github.com/2LTech/nextjs-app-passport).

Used functions:

- `setLocalStrategy`
- `APILoginRoute`
- `APILogoutRoute`
- `getSession`

:warning: Users managment used in this project is for demonstration purpose only, you should use, for example, a database to create and store your users.
